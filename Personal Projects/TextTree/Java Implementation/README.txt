
TextTree GUI Application (Java)

Still in progress as of Jan. 2017

Functionality currently supported:

    Underlying data structure TextTree, fully functional 
    Window, with toolbar, text fields, buttons, list and tree structure GUI components in place
    All GUI components functional except for the toolbar menu items, some bugs in the list
    Adding and removing nodes to both the TextTree and the JTree
    Editing the labels and content fields of the nodes in the TextTree and JTree
    Node children displayed in list

Major functionality yet to implement:

    File structure for loading and saving
    Read and write methods for loading and saving
    Toolbar menu items and their operations (clear, restore previous state, exit, save, load, etc.)
