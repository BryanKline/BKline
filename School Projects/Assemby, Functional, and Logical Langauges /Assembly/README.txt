Assembly code sample produced for school assignment.

Detailed:

The code provided here, , split between three files, is a sample of code 
produced as an assignment for CS219, Computer Organization, to implement
some C code in the MIPS assembly language.  The purpose is to demonstrate 
some exposure to an assembly langauge and the concepts of lower level
programming in general.

In addition to the files containing the code the assignment itself is included
which can be consulted to provide context to the solutions.
