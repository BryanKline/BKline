
Assembly, Functional and Logical Languages

Brief:

Folders contain samples of assembly, Scheme, and Prolog code.

Detailed:

Folders contain code samples from either a lower level language or different
programming paradigms, (assembly, functional, and logical languages) produced
as assignments for CS219 and CS326, Computer Organization and Programming
Languages, Concepts, and Implementation, respectively.

Each folder contains code samples and and assignment description for which
the code was written.
